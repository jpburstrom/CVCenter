CVWidget the abstract superclass for various widgets like CVWidgetKnob, CVWidget2D (allowing to set CVs in a Slider2D resp. a RangeSlider) and CVWidgetMS (allowing to set CVs, containing multidimensional ControlSpecs via a MultiSliderView - not yet implemented)


inherits from: Object

dependencies: Conductor (Quarks)


A CVWidget is not meant to be used directly. Basically it holds properties common to all CVWidget-subclasses. So far CVWidgetKnob and CVWidget2D have been implemented.
